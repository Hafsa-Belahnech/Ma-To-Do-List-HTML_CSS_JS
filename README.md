# Simple To-Do List

A minimalist and functional To-Do List application built with vanilla web technologies. This project helps you manage your daily tasks through a clean and intuitive interface.

## 🚀 Features
- **Add Tasks**: Quickly create new items for your list.
- **Mark as Done**: Toggle task completion with a single click.
- **Delete Tasks**: Remove items from your list instantly.
- **Data Persistence**: Uses `localStorage` to keep your tasks saved after refreshing the page.

## 🛠️ Technologies Used
- **HTML5**: Application structure.
- **CSS3**: Custom styling and layout.
- **JavaScript (ES6)**: Task management logic and DOM manipulation.
